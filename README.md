Agenda de Contactos

Proyecto realizado en Java utilizando Programación Orientada a Objetos.

Funciones:

✔ Agregar contactos
✔ Buscar contactos
✔ Eliminar contactos
✔ Listar contactos
✔ Validación de correo
✔ Validación de teléfono
✔ Evita duplicados
✔ Persistencia de datos mediante archivo TXT

Tecnologías:

- Java
- Programación Orientada a Objetos
- ArrayList
- Scanner
- Manejo de archivos (IO)
